Проект: Статистический анализ данных сервиса аренды самокатов

Стек: python (pandas, numpy, matplotlib, seaborn, scipy)

Cервис предоставляет услуги по аренде самокатов в нескольких городах РФ. Пользователи могут пользоваться сервисом без подписки (Free) и с подпиской (Ultra). Цель проекта: оценить целесообразность увеличения количества пользователей с подпиской.

Исходные данные: три датасета с информацией о пользователях, их поездках и подписках из нескольких городов РФ

План исследования:

1. загрузка и предобработка данных (проверка и обработка дубликатов и пропусков, приведение типов данных к нужным форматам)
2. исследовательский анализ, в т.ч. группировка пользователей и расчет выручки по группам
3. проверка нескольких гипотез
4. расчет показателей: расчет мин.количества промокодов и оценка вероятности открытия push-уведомлений


Вывод:  среднемесячная выручка у пользователей с подпиской больше, чем у пользователей без подписки, разница статистически значима. Т.о. целесообразно увеличивать пользователей с подпиской.
