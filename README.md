### 2.	Анализ данных продаж интернет-магазина

Цель проекта: проанализировать результаты проведенного А/В теста, проверить гипотезы о наличии статистически значимых различий в метриках между двумя группами покупателей (А и В) и сделать вывод о завершении теста и победе одной из групп или продолжении тестирования.
Описание проекта:  рассчитаны и проанализированы кумулятивные данные по выручке, чеку, количеству заказов на посетителя по группам,  данные проверены на наличие выбросов и аномалий, проверена статистическая значимость различий в среднем чеке заказа и количеству заказов на пользователя по группам.
Вывод: тест следует остановить, зафиксировать победу группы В.

### 3.	Анализ оттока клиентов банка

Цель проекта: проанализировать клиентов одного регионального банка, выделить сегменты клиентов, которые наиболее склонны уходить из банка, чтобы предложить рекомендации по их удержанию.
Описание проекта:  проведена предобработка и анализ данных , построена матрица корреляций оттока с остальными признаками, сформировать портреты клиентов в разрезе оттока, проведена сегментация клиентов, проверено несколько гипотез.   
Вывод: выделены категории клиентов, более склонных к оттоку (уровень оттока выше среднего по банку), предложены рекомендации по их удержанию.

### 4.	Маркетинговый анализ игрового приложения

Цель проекта: выяснить причины убыточности игрового приложения. В бизнес-плане заложено, что пользователи должны окупаться не позднее чем через две недели после привлечения.
Описание проекта:  проанализированы пользователи в разбивке по регионам, устройствам, каналам привлечения, сформированы пользовательские профили. Рассчитаны и проанализированы ключевые метрики: удержание, конверсия, LTV, CAC, ROI в разбивке по регионам, устройствам, каналам привлечения. Построены графики. 
Вывод:  основная проблема - это низкое удержание и неокупаемость пользователей из США, которая обусловлена высокой стоимостью двух каналов TipTop и FaceBoom (ключевые источники трафика в США). Следует проанализировать условия работы с данными каналами, выяснить причину резкого роста расходов на привлечение.

### 5.	Статистический анализ данных сервиса аренды самокатов

Цель проекта:  проанализировать  данные о клиентах сервиса аренды самокатов  и оценить целесообразность увеличения количества пользователей с подпиской.
Описание проекта:  проведена предобработка и анализ данных, построены диаграммы, боксплоты, рассчитаны показатели  по группам пользователей с подпиской и без, проверено несколько гипотез. 
Вывод: анализ показал, что  среднемесячная выручка у пользователей с подпиской больше, чем у пользователей без подписки, при этом разница статистически значима. Т.о. целесообразно увеличивать пользователей с подпиской.
