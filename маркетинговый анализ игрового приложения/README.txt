Маркетинговый анализ игрового приложения
стек: python (pandas, seaborn, matplotlib, numpy)

Цель исследования: выяснить причины убыточности игрового приложения. В бизнес-плане заложено, что пользователи должны окупаться не позднее чем через две недели после привлечения

Исходные данные: 
лог сервера с данными об их посещениях,
выгрузка их покупок за этот период,
рекламные расходы.

План исследования:
1. загрузить данные
2. выполнить предобработку данных:
3. проверить, обработать пропуски
4. проверить на наличие явныx и неявных дубликатов
5. проверить корректность форматов данных, названий полей и т.д.
6. провести исследовательский анализ данных:
7. проанализировать пользователей в разбивке по регионам, устройствам, каналам привлечения
8. сформировать пользовательские профили
9. рассчитать и проанализировать ключевые метрики: удержание, конверсия, LTV, CAC, ROI в разбивке по регионам, устройствам, каналам привлечения
10. проанализировать маркетинговые расходы
11. сформулировать выводы по итогам анализа


Выводы по проекту: 
Основная проблема - это низкое удержание и неокупаемость пользователей из США, которая обусловлена высокой стоимостью каналов TipTop и FaceBoom (ключевые источники трафика в США). Следует проанализировать условия работы с данными каналами, выяснить причину резкого роста расходов на привлечение.

США - основной рынок, при этом пользователи из США не окупаются в течение 14 дней (заложенный бизнес-планом срок), несмотря на самый высокий LTV среди всех стран. Причина - высокая стоимость привлечения пользователей из США, в т.ч. резкий рост расходов на привлечение начиная с июня месяца, что обусловлено ростом расходов каналов TipTop и FaceBoom.
Данные каналы являются наиболее дорогостоящими каналами привлечения: 52% и 31% всех расходов на рекламу соответственно, именно они являются основной причиной совокупного роста расходов на рекламу и неокупаемости пользователей.
